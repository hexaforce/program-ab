Class Overview

The core AIML parser and interpreter. Implements the AIML 2.1 specification as described in AIML 2.1 Working Draft document https://docs.google.com/document/d/1wNT25hJRyupcG51aO89UcQEiG-HkXRXusukADpFnDs4/pub
